# Spotify-Backend
A scalable and secure backend system inspired by modern music streaming platforms, built using Node.js, Express, MongoDB, and JWT Authentication.
This project demonstrates backend architecture, authentication, file handling, API development, and cloud media integration.

🚀 Features

✅ User Authentication & Authorization
✅ JWT-based Secure Login System
✅ Password Hashing with Bcrypt
✅ RESTful API Architecture
✅ MongoDB Database Integration
✅ Image Upload Handling with Multer
✅ Cloud Media Management using ImageKit
✅ Environment Variable Security
✅ Modular & Scalable Backend Structure
